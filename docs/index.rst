=========
auto-acmg
=========

Auto-acmg is a program for automatical prediction of PVS1 ACMG criterion for given variant.
The algorithm is based on the paper `Recommendations for interpreting the loss of function PVS1 ACMG/AMP variant criterion <https://www.ncbi.nlm.nih.gov/pmc/articles/PMC6185798/>`__.
If you want to **use** auto-acmg, the best place is to start at :ref:`dev_quickstart`.
